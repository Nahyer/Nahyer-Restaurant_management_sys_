Welcome to Our API

Thank you for signing up! Here are some instructions on how to use our API endpoints:

Sign Up

To create a new account, send a POST request to the following endpoint:

POST http://localhost:3000/auth/sign_up

Request Body Example:

{
    "name": "Lee",
    "contact_phone": "1234567890",
    "phone_verified": false,
    "email": "lee@example.com",
    "email_verified": false,
    "confirmation_code": "",
    "username": "lee_username",
    "password": "your_password",
    "role": "User"
}

Upon successful sign-up, you will receive a verification email. Please follow the instructions in the email to verify your account.

Email Verification

To verify your email, click on the verification link sent to your email. The link will look like this:

http://localhost:3000/auth/verify_email?code=your_confirmation_code

This will update your email verification status in our system.

Login

Once your email is verified, you can log in by sending a POST request to the following endpoint:

POST http://localhost:3000/auth/login

Request Body Example:

{
    "username": "lee_username",
    "password": "your_password"
}

Additional Endpoints

Here are some additional endpoints you can use:

  • GET /users: Retrieve a list of users.
  • GET /users/:id: Retrieve a specific user's information.
  • PUT /users/:id: Update a user's information.
  • DELETE /users/:id: Delete a user.

Replace :id with the actual user ID.
